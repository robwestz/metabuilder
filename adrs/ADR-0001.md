
# ADR-0001: PSIR i TypeScript, API i Python

- **Beslut**: PSIR-parsing/transform i TypeScript; API i FastAPI (Python).
- **Skäl**: Snabb toolchain, delbar kod till web/CLI; Python stabilt för API.
- **Alternativ**: Allt i Python; allt i Node.
- **Konsekvens**: Två körmiljöer; tydlig separering och oberoende deployment.
